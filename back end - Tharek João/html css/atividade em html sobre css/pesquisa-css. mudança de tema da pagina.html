📜 História do CSS

O CSS surgiu em 1996 para separar o conteúdo do HTML da parte visual das páginas. Antes disso, o HTML misturava estrutura com estilos, deixando o código confuso. O CSS trouxe organização, reaproveitamento de estilos e layouts mais modernos.

👨‍💻 Quem inventou o CSS?

O CSS foi criado por Håkon Wium Lie junto com Bert Bos, enquanto trabalhavam no World Wide Web Consortium (W3C).

✅ Vantagens do CSS

  • Separa conteúdo da aparência
  • Deixa o site mais organizado
  • Facilita manutenção
  • Permite layouts modernos e responsivos
  • Reaproveitamento de estilos
